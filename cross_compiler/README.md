Building Your own Cross compiler for your own OS
-------

Step 1:
  
Run `make clonedeps`

Step 2:

Run `make unzip`

Step 3

Run `make build`

after the build completes you should see a `cross/` folder with the cross compiler

Use the command `source sourcecc.sh` file to add it to your path
